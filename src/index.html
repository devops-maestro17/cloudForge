PassShield-K8s

A comprehensive password security app that evaluates password strength and checks for compromises using third-party API, featuring a React frontend and FastAPI backend containerized with Docker. E-Commerce-Sentinel

A comprehensive CI/CD pipeline for deploying an Amazon clone application to an Elastic Kubernetes Service (EKS) cluster, following DevSecOps best practices. The infrastructure is provisioned using Terraform, and Jenkins is utilized as the Continuous Integration (CI) tool. The Continuous Deployment (CD) stage is leveraged using ArgoCD, to automatically deploy the updated application to the EKS cluster.

Documentation & Source Code

MultiKubeDeploy

Automating the deployment of an application to multiple Google Kubernetes Engine (GKE) clusters using ApplicationSets in ArgoCD. ArgoCD is utilized to manage multiple clusters via a hub-spoke architecture, where a centralized hub cluster hosts the ArgoCD instance responsible for managing resources across multiple spoke clusters. The Kubernetes manifests of the application are packaged using Helm charts, and the Kubernetes clusters are provisioned using Terraform within a custom Virtual Private Cloud (VPC).

Documentation & Source Code

EnvHorizon

Deploying a scalable and secure AWS infrastructure using Terraform. It leverages modularity and flexibility through Terraform modules. The infrastructure includes a VPC with public and private subnets, EC2 instances, a load balancer, and an RDS database. The project also incorporates best practices like remote state storage, multiple environments, static checks, state locking, and secrets management using HashiCorp Vault.
